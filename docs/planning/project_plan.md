## 1. 서비스 개요

### 1.1 서비스 컨셉

"Find My Rizz"는 사용자가 자신의 잠재적 매력(Rizz)을 발견하도록 돕는 **AI 자기관리 시각화 서비스**입니다.

과장된 뷰티 필터나 비현실적인 변형이 아닌, **'체계적인 관리를 통해 도달 가능한'** 가장 자연스럽고 매력적인 모습을 AI가 시각화하여 3장의 이미지로 제공합니다.

### 1.2 서비스 목표

1. **동기 부여:** 사용자에게 '관리된 나의 미래 모습'을 미리 보여줌으로써, 자기 관리의 명확한 목표와 강력한 동기를 부여합니다.
2. **신뢰 구축:** '왜곡 없음'과 '현실적인 개선'을 원칙으로 하여, 기존 AI 프로필 서비스와 차별화된 신뢰를 구축합니다.
3. **간편한 경험:** 회원가입과 앱 설치 없이, 웹에서 결제하고 SMS로 결과를 받는 간편한(Frictionless) 경험을 제공합니다.

### 1.3 타겟 고객

- **핵심 타겟:** 자기 관리에 관심은 많지만, 명확한 목표나 동기가 필요한 20~30대 남녀
- **확장 타겟:** 기존 뷰티 앱의 과장된 필터에 피로감을 느끼며 '현실적인 개선'에 대한 니즈가 있는 사용자

## 3. 사용자 시나리오 (User Flow)

1. **방문:** 사용자가 서비스 소개 페이지를 보고 '지금 바로 나의 리즈 알아보기' 버튼을 클릭합니다.
2. **결제 시도:** 결제창이 열리고 9,900원 결제를 진행합니다.
   - **[실패 시]** "결제에 실패했습니다" 알림 후, 서비스 소개 페이지로 돌아갑니다.
3. **결제 성공:** 결제가 완료되면 전화번호 입력 페이지로 이동합니다.
4. **전화번호 입력:** 결과를 받을 휴대폰 번호(필수)를 입력합니다.
5. **사진 가이드 확인:** "최고의 결과를 위한 사진 가이드" (O/X 예시)를 확인합니다.
6. **사진 업로드 및 편집:** 갤러리/카메라에서 사진을 선택한 후, **3:4 크롭 도구**를 이용해 사용자가 직접 사진을 편집하고 "확인" 버튼을 누릅니다.
7. **신청 완료:** "신청이 완료되었습니다. 생성이 완료되면 문자로 안내드립니다."라는 안내 페이지를 봅니다. (사용자는 이 창을 닫아도 됩니다.)
8. **결과 수신 (SMS):** 사용자의 휴대폰으로 **고유한 결과 URL**이 포함된 SMS가 전송됩니다.
9. **결과 확인:** 사용자가 SMS의 URL을 클릭하여 결과 페이지로 접속합니다.
10. **전화번호 인증:** 보안을 위해 전화번호를 다시 입력하여 인증합니다.
11. **결과 조회 및 저장:** "24시간 후 자동 파기" 안내와 함께, AI 이미지 3장과 맞춤형 리포트를 확인합니다. '내 사진 모두 다운로드' 버튼으로 24시간 내에 이미지를 다운로드합니다.

## 4. 향후 고도화 방향 (Roadmap)

### 4.1 LangGraph 기반 아키텍처

- **현재 구현:** MVP부터 LangGraph를 활용한 확장 가능한 아키텍처로 개발합니다.
- **목표:** 복잡한 AI 워크플로우를 체계적으로 관리하고, 향후 기능 확장 시 유연하게 대응합니다.
- **적용 영역:**
  - 이미지 분석 및 리포트 생성 파이프라인
  - 관리법 매칭 및 추천 로직
  - 다단계 AI 처리 워크플로우
- **장점:**
  - 상태 관리 및 에러 복구 용이
  - 각 단계별 독립적 테스트 가능
  - 새로운 AI 기능 추가 시 기존 코드 영향 최소화

### 4.2 맞춤형 리포트 고도화

- **목표:** 기본 리포트를 사용자의 특성에 맞춘 '개인화 리포트'로 발전시킵니다.
- **내용:** AI가 생성한 이미지 분석을 기반으로, 사용자에게 필요한 **자기관리 요소**(예: 피부 시술, 헤어 스타일링)를 제안합니다.
- **데이터:** '자기관리 요소 테이블'(설명, 평균 금액대 등)을 내부 데이터로 구축하여 AI 리포트 생성에 활용합니다.

### 4.3 스타일 선택 기능

- **목표:** 사용자가 원하는 결과물의 분위기를 직접 선택하도록 하여 만족도를 높입니다.
- **내용:** 사진 업로드 시 '전문적인', '부드러운', '자신감 있는' 등 원하는 스타일 키워드를 선택하게 하고, 이를 AI 이미지 생성에 반영합니다.
